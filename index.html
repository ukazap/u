Proof of concept URL shortener with no back-end servers (apart from GitHub's!).

  1. Create an issue on the ukazap/u repository. The title of the issue should be the URL you want to redirect to.

  2. Note n, the issue's number.
  3. Construct your shortened URL using n: https://ukazap.github.io/u/?n

Correct URLs are in the format https://ukazap.github.io/u/?n, where n is an issue on the ukazap/u repository. The title of the issue is used as the URL to redirect.
